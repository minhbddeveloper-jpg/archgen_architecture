# arxgen v1.9.0

## Highlights

- Added `--auth-mode production` for TypeScript Express JWT auth generation.
- Added generated auth config that requires `JWT_SECRET` in production auth mode.
- Added refresh-token hash storage, token rotation, revoke, refresh, and logout flow.
- Added generated RBAC repository contracts and Prisma auth models for users, refresh tokens, roles, permissions, user roles, and role permissions.
- Added focused production auth generation tests.

## Migration Notes

Existing generated projects are not modified automatically. New projects can opt into production auth with:

```bash
arxgen create --auth jwt --auth-mode production ...
```

Generated production auth requires `JWT_SECRET` at runtime.

## Breaking Changes

None for existing CLI commands. Production auth mode is opt-in.
