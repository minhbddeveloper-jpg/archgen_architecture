# arxgen v1.6.0

## Highlights

- Promoted NestJS generation with build-verified project output.
- Added `npm run test:e2e:nestjs` for generated NestJS install/build/start/CRUD verification.
- Added a generated NestJS health endpoint.
- Added NestJS Prisma repository generation when using `--orm prisma`.
- Added Prisma package setup for generated NestJS Prisma projects.
- Extended `arxgen add entity` to generated NestJS projects and app module registration.

## Migration Notes

Regenerate NestJS projects to receive the new health controller, production start script, async CRUD services/controllers, and Prisma repository support.

## Breaking Changes

None.
