# arxgen v1.5.0

## Highlights

- Express projects generated with `--database postgres --orm prisma` now use Prisma-backed CRUD repositories.
- SQL schema import can generate entities, Prisma schema models, relations, and CRUD routes from `--from-sql`.
- Generated TypeScript Express README files now include setup, development, build, database, migration, API example, and environment sections.
- `arxgen doctor` now checks Node.js, npm, output folder write access, package manager context, Docker availability, and database configuration.

## Migration Notes

Existing generated projects can keep using in-memory repositories. Regenerate or add entities in a Prisma-enabled project to get database-backed repositories.

## Breaking Changes

None.
