# arxgen v1.4.0

## Changed

- Refactored project generation through `GenerationPipeline` and `FeatureGenerator`.
- Updated README to separate stable, partial, and scaffold capabilities.
- Split long README content into focused docs.
- Added plugin capability reporting to `arxgen list plugins`.
- Updated roadmap status to match the package version and current implementation.

## Added

- Added generated Express app execution test scaffold in `tests/express-generated-app.test.mjs`.
- Added `npm run test:e2e` for generated Express app install/build/start/CRUD verification.
- Added docs for getting started, CLI reference, supported stacks, SQL import, schema upgrade, plugin development, and production readiness.
- Added SQL metadata parsing for unique/default/length/precision information.

## Hardened

- Generated JWT scaffold now rejects missing `JWT_SECRET` in production.
- Generated auth middleware no longer hard-codes user permissions.

## Validation

- Default test suite: `npm test`.
- Generated app e2e test: `npm run test:e2e`.
