# arxgen v1.3.0

## Added

- Added `arxgen upgrade schema --from-sql <file>` for generated TypeScript Express projects.
- Added schema diff support for additive SQL changes.
- Added dry-run preview output for schema upgrades.
- Added safe patching for:
  - domain entity fields
  - validation schemas
  - create use cases
  - Prisma models
- Added automatic creation of new entities found in the upgraded SQL schema.

## Safety Rules

- Schema upgrade does not delete fields.
- Schema upgrade does not rename fields automatically.
- Destructive database changes must still be handled manually.

## Validation

- Added tests for dry-run preview and real schema upgrade behavior.
- Verified with `npm test`.
