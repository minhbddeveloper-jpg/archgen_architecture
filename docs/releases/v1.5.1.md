# arxgen v1.5.1

## Highlights

- Completed the v1.5 documentation cleanup and roadmap status updates.
- Added PostgreSQL-backed Express + Prisma e2e coverage through `npm run test:e2e:postgres`.
- Added configurable generated Docker Compose database ports for PostgreSQL, MySQL, and MongoDB.
- Renamed the npm package to the scoped package `@minhbddeveloper/arxgen` for public publishing.

## Migration Notes

No application migration is required. Regenerate projects to receive the updated Docker Compose port variables.

None.
